Making a blog

Here's how this devlog was created.

The goal is to make a minimal static site generator for a devlog. We want to write a pipeline that takes in a Markdown file and some frontmatter. That Markdown is converted to HTML. We then render into a template. This output is then written to HTML files. Aftewards we generate an index page holding all the posts. Finally, we generate an RSS/Atom Feed.
