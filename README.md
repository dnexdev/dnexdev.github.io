# Static Site Generator for devlog

Built using jinja and python.

# How to use

1. Clone the repository.
2. Create a virtual environment and install all libraries in requirements.txt
3. To create a post, create a folder where you store all the assets in your post.
4. Write in markdown that markdown2 can process. 
5. First run python build.py in order to build the files
6. To preview what the website looks like, cd docs && python3 -m http.server 8000
